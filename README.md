# Pharmapp

This is a SpringBoot application called **Pharmapp**. It is an application that displays most of the pharmacies in Macedonia.
When clicked on, each pharmacy displays it's details such as: name, location, website, phone number, and working hours.

Application functionalities:
- Displaying the pharmacies and details as well as plotting them on an interactive map.
- Finding the user's location and using it to navigate the user to the closest pharmacy.
- Displaying a route on the map, as well as listing directions to a closest pharmacy, or a pharmacy of choice.
- Adding a new pharmacy (Only possible if a user is logged in).
- Creating new users.
